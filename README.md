# Trend Application - DevOps CI/CD Project

## Project Overview
Full-stack React application deployed on AWS EKS using Jenkins CI/CD pipeline.

## Architecture
- **Frontend**: React application
- **Container**: Docker
- **Registry**: AWS ECR
- **Orchestration**: Kubernetes (AWS EKS)
- **CI/CD**: Jenkins
- **Monitoring**: Prometheus & Grafana
